# 公共看板能力

这个功能区是价格页和库存上下架页共用的底座。

## 运行入口

- 服务入口：`src/server.js`
- 前端入口：`public/index.html`
- 前端公共逻辑：`public/app.js`
- 数据库连接：`src/db.js`
- 表结构：`src/schema.js`
- 快照读写、备注、操作记录：`src/snapshot-store.js`

## 公共能力

- 登录操作人
- 操作记录
- 单条备注、批量备注
- 单条认领、批量认领
- 未处理、已完成、弃用状态
- SKU-运营映射表上传
- 从 MySQL 读取最新 `price` / `inventory` 快照

## 修改原则

只改价格显示时，尽量不要动这里。只有登录、备注、操作记录、数据库结构这类两个页面都共用的能力，才归这里。
